Installation Guide

For Windows Users

Follow these steps to install LinguaGenie on your Windows machine:

  1. Step 1: Allow Script Execution

    Open PowerShell as Administrator and run the following command to allow script execution:

    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  2. Step 2: Download the Installer Script

    Run the following command to download the installation script:

    Invoke-WebRequest -Uri "https://raw.githubusercontent.com/ZAYUVALYA/LinguaGenie/main/INSTALL/install.ps1" -OutFile "$env:TEMP\install.ps1"
  3. Step 3: Run the Installer

    Execute the downloaded script to start the installation process:

    & "$env:TEMP\install.ps1"

For Linux Users

Follow these steps to install LinguaGenie on your Linux machine:

  1. Step 1: Download the Installer Script

    Use the following command to download the installation script:

    wget -O "$HOME/install.sh" "https://raw.githubusercontent.com/ZAYUVALYA/LinguaGenie/main/INSTALL/install.sh"
  2. Step 2: Make the Script Executable

    Grant execution permissions to the script by running:

    chmod +x "$HOME/install.sh"
  3. Step 3: Run the Installer

    Execute the script to begin the installation:

    bash "$HOME/install.sh"

Important Notes:

  • Ensure you have a stable internet connection during the installation process.
  • For Windows, running PowerShell as Administrator is required to execute the script.
  • For Linux, ensure you have wget installed. If not, install it using your package manager (e.g., sudo apt install wget).
  • If you encounter any issues, please refer to the Report a Bug section.
